# 🍽️ Сайт ресторана "BAI KAZAN"

## 📖 Описание
Это веб-приложение ресторана для доставки блюд на дом.  
🌐 Сайт доступен по ссылке:  
👉 https://Maison-Mardjuha-13.github.io/Restaurant

---

## ⚙️ Функционал

Реализованы следующие возможности:

- 📋 Меню с возможностью добавления блюд в корзину  
- 🛒 Корзина (добавление и удаление товаров)  
- ✅ Оформление заказа  
- 📅 Бронирование стола  
- 🚚 Страница с информацией о доставке  
- 💼 Страница с вакансиями  
- 🎯 Лендинг  

---

## 🛠️ Технологии

Проект разработан с использованием:

- **Frontend:** Vue.js 3, Vuex  
- **Backend:** Python (FastAPI)  
- **Database:** PostgreSQL
